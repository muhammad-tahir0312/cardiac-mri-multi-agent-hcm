# Cardiac MRI Multi-Agent System for HCM Analysis

> Automated decision support for Hypertrophic Cardiomyopathy diagnosis using multi-agent deep learning on cardiac MRI.

## 📋 Dataset
- Source: [Kaggle - Hypertrophic Cardiomyopathy Dataset](https://www.kaggle.com/datasets/danialsharifrazi/hypertrophic-cardiomyopathy-dataset)
- Format: DICOM → NIfTI conversion required
- Task: Binary classification (HCM vs Healthy) + optional segmentation
